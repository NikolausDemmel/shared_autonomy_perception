KeyboardTeleop

new KeyboardTeleop(options)

Manages connection to the server and all interactions with ROS. Emits the following events: * 'change' - emitted with a change in speed occurs
Parameters:
Name Type Description
options possible keys include: * ros - the ROSLIB.Ros connection handle * topic (optional) - the Twist topic to publish to, like '/cmd_vel' * throttle (optional) - a constant throttle for the speed
